# We Remember

**A platform for things worth remembering.**

We Remember is an API-first, self-hostable platform designed to help preserve, connect, and revisit meaningful information across time.

The long-term vision is simple:

> Help people remember what matters.

Built for developers, families, communities, and anyone who believes important things should not be forgotten.

## Principles

* API First
* Self Hostable
* Extensible by Design
* Open Source
* Built for the Long Term

## Why?

Technology should help people remember what matters.

In a world of information overload, it's easy to lose sight of the people, events, traditions, and experiences that shape our lives.

We Remember exists to help individuals, families, and communities preserve meaningful information across time — not just to store it, but to reconnect with it when it matters most.

Because some things are worth remembering.
